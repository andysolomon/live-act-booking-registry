# live-act-booking-registry

Marketplace scaffold for booking live performers with verified reliability, crowd-draw history, and venue feedback.

## Stack
- Next.js (App Router) + TypeScript
- Bun
- Tailwind CSS

## Getting Started
1. `bun install`
2. `cp .env.example .env.local`
3. `bun run dev`

## Scripts
- `bun run dev`
- `bun run build`
- `bun run start`
- `bun run lint`
- `bun run typecheck`
- `bun run test`
